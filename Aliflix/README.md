# Aliflix - Film Öneri Sistemi

Aliflix, React + TypeScript + SASS ile geliştirilen modern bir film öneri sistemidir.

## 🚀 Özellikler

- React 18 + TypeScript
- SASS/SCSS styling
- Feature-based mimarisi
- Modern UI/UX
- Responsive tasarım
- Custom hooks
- Context API
- API Services

## 📁 Proje Yapısı

```
src/
├── features/        # Feature-based components
├── components/      # Reusable components
├── hooks/          # Custom React hooks
├── services/       # API services
├── context/        # React Context
├── animations/     # Animation utilities
├── styles/         # Global SCSS styles
├── utils/          # Utility functions
├── types/          # TypeScript types
├── constants/      # Constants
└── App.tsx         # Main App component
```

## 🛠️ Kurulum

```bash
npm install
```

## 🚀 Geliştirme Sunucusu

```bash
npm run dev
```

## 📦 Build

```bash
npm run build
```

## 👁️ Preview

```bash
npm run preview
```

## 📝 Notlar

- Feature-based mimarisi ile modüler yapı
- Her feature kendi klasöründe organize edilir
- Shared components `components` klasöründe
- Global styles `styles` klasöründe
- Type safety için TypeScript kullanılıyor
- SASS kullanarak temiz ve maintainable CSS
